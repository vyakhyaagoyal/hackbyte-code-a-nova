# 🧠 SelfSync AI — Intelligent Study Room

**AI-powered study environment that improves focus, reduces distractions, and provides real-time AI tutoring + personalized performance feedback.**

Built at **HackByte 4.0** under theme **Patch the Reality**.

---

## 🚀 Key Features

### 🎯 AI Study Monitoring
- Detects **distraction** (looking away from screen)
- Detects **phone usage** using YOLOv8
- Detects **yawning** (fatigue indicator)
- Tracks **blink intervals** for eye strain analysis
- Calculates **% of session distracted**
- Logs **focus behaviour timestamps**

### 🎙️ AI Tutors (Voice Enabled)
- Real-time conversational tutors using **ElevenLabs**
- Ask doubts without leaving study environment
- Multilingual voice interaction
- Reduces tab switching → improves productivity

### 📊 AI Performance Report
- Generated using **Google Gemini API**
- Provides **focus rating + improvement suggestions**
- Downloadable **PDF report**
- Converts behavioural data into actionable insights

### ⏱️ Productivity Tools
- **Start Session** → webcam monitoring starts
- **End Session** → monitoring stops + report downloads
- **Pomodoro focus timer**
- **White noise** for distraction-free studying
